# Expense Tracker

A full-stack expense tracking application built with Node.js, Express, SQLite, and vanilla JavaScript.

## Features
- Add expenses with name, amount, and category
- View all expenses with running total
- Delete individual expenses
- Data persists in SQLite database

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite3

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `node server.js`
4. Open `http://localhost:3000`

## What I Learned
- Building REST APIs with Express
- SQLite database operations (CRUD)
- Connecting frontend to backend with fetch API
- Handling CORS for local development
